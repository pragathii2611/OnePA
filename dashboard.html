// SETUP GUIDE

1
Install Python dependencies:
pip install -r requirements.txt
2
Copy .env.example to .env and fill in your Gmail and facility IDs.
3
Create a Gmail App Password at myaccount.google.com → Security → 2-Step → App passwords and paste it into .env.
4
Run the monitor:
python monitor.py
It checks every 5 minutes and emails you the moment slots appear.
5
Pro tip: New slots open at 10 PM daily. You can lower CHECK_INTERVAL_MIN=1 around that time for fastest alerts.
